# JavaScript Performance Optimization: Advanced Techniques

Performance is crucial in modern web applications. Learn how to write more efficient JavaScript code.

## Memory Management

### Avoiding Memory Leaks

```javascript
// Bad
function createLargeArray() {
  const largeArray = new Array(1000000);
  // Do something
  return largeArray;
}

// Good
function createLargeArray() {
  const largeArray = new Array(1000000);
  // Do something
  return largeArray.slice(); // Creates a copy
}
```
