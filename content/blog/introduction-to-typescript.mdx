# TypeScript Fundamentals: Beyond JavaScript

TypeScript extends JavaScript by adding type safety and advanced object-oriented programming features.

## Basic Type Annotations

```typescript
// Type annotations
let name: string = "John Doe";
let age: number = 30;
let isStudent: boolean = false;

// Function types
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

## Advanced Type Features

### Interfaces

```typescript
interface User {
  id: number;
  name: string;
  email: string;
  age?: number; // Optional property
}
```

### Generics

```typescript
function identity<T>(arg: T): T {
  return arg;
}
```

## Benefits of TypeScript

1. **Type Safety**
2. **Enhanced IDE Support**
3. **Easier Refactoring**
4. **Improved Code Documentation**

## Conclusion

TypeScript offers powerful tools to write more robust and maintainable JavaScript code. Embrace the type system!
